# homebrew-keel

Homebrew tap for [keel](https://github.com/joeymiller/keel) — controlled execution governance for AI coding agents.

## Install

```bash
brew tap joeymiller/keel
brew install keel
```

## Upgrade

```bash
brew update
brew upgrade keel
```

## Uninstall

```bash
brew uninstall keel
brew untap joeymiller/keel
```

---

*Formula is updated automatically by the keel release workflow.*
